# 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de **Java** e **Programação Orientada a Objetos**.

O sistema apresenta um **menu interativo no terminal** contendo diversas funcionalidades relacionadas a cálculos e conversões.

---

# 🚀 Funcionalidades

✔ Calculadora básica  
✔ Cálculo de IMC  
✔ Cálculo de área do triângulo  
✔ Conversão de temperatura  
✔ Conversão de moeda  
✔ Operações bit a bit  

---

# 🛠 Tecnologias Utilizadas

- ☕ Java
- 🧠 Programação Orientada a Objetos (POO)
- ⌨ Entrada de dados com Scanner
- 🧩 Estrutura modular de código

---

# 📂 Estrutura do Projeto

```
src/
├─ entities/
│  └─ CalculoGeral.java
│
└─ project/
   ├─ MenuPrincipal.java
   └─ Programa.java
```

### 📌 Descrição das Classes

**CalculoGeral.java**  
Responsável pelos cálculos e lógica das operações do sistema.

**MenuPrincipal.java**  
Controla o menu interativo exibido no terminal.

**Programa.java**  
Classe principal responsável por iniciar o sistema.

# ▶ Como Executar o Projeto

### 1️⃣ Clone o repositório

```
git clone https://github.com/PauloHMJ/NOME-DO-REPOSITORIO
```

```

```

### 3️⃣ Execute o programa

